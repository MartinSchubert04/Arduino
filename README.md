## About

### Setup for wokwi (arduino simulator) and some projects

## Dependecies

- Arduino-CLI for windows
- Wokwi extension for VS code

## Intall dependencies
```pswh
arduino-cli lib install "lib name"
```

## Compile
```pswh
arduino-cli compile --fqbn arduino:avr:uno --output-dir ./build ./pathToFile.ino
```
